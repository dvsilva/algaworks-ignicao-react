# algaworks-ignicao-react

- criar projeto do curso
  npx create-react-app alga-stock --template typescript
- iniciar projeto
  cd alga-stock
  npm start ou yarn start

Tecnologias

- React
- Sass
- Swal
- Fetch API
- Axios
- Prism
- Redux (thunk e persist)
- React Router DOM
- HOC (High Order Component)
- Deploy Netlify
- Pagination
